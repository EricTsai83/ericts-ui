## Introduction

ericts/ui is a custom shadcn registry for installable components, hooks, and
blocks.

Use these docs to find registry items, preview their behavior, and copy the
install command for your project.

Key routes:

- `/docs`
- [Components](/components)
- [Hooks](/hooks)
- [Blocks](/blocks)
- `/view/base/:name`
- `/api/search`
- `/r/:name.json`

## Installation

Install an item with the shadcn CLI and the public registry JSON endpoint.

```bash
pnpm dlx shadcn@latest add https://ui.ericts.com/r/copy-button.json
```

Replace `copy-button` with any item name from the components, hooks, or blocks
pages. See [Installation](/docs/installation) for namespace details and local
registry build notes.
