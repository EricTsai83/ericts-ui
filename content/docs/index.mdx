ericts/ui is a shadcn-compatible registry for interface details that should feel
deliberate: motion primitives, interaction patterns, focused hooks, and polished
components you can install as source.

Use this site to browse the available items, preview how they behave in real
UI states, and add only the pieces you need to an existing shadcn project. Each
registry item is designed to be readable, adaptable, and easy to own once it
lands in your codebase.
