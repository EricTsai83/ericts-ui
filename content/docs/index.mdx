## Overview

ericts/ui is a custom shadcn registry for installable components, hooks, and blocks.

The registry JSON is published from `/r/:name.json`, and every item has a
preview route under `/view/base/:name`.

## Public routes

- `/docs`
- `/components`
- `/hooks`
- `/blocks`
- `/view/base/:name`
- `/api/search`
- `/r/:name.json`
