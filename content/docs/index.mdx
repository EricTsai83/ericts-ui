## Overview

EricTS UI is a custom shadcn registry for installable components and blocks.

The registry JSON is published from `/r/:name.json`, and every item has a
preview route under `/view/new-york/:name`.

## Public routes

- `/docs`
- `/blocks`
- `/view/new-york/hello-world`
- `/api/search`
- `/r/hello-world.json`
